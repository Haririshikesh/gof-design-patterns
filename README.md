# GoF Design Patterns Reference Vault

Welcome! This repository is a lightweight, text-based cheat sheet for the 23 classic **Gang of Four (GoF) Design Patterns**. 

## 🗺️ Navigation

* **[Creational Patterns](./creational-patterns/)**: Dealing with object creation mechanisms.
* **[Structural Patterns](./structural-patterns/)**: Dealing with object composition and relationships.
* **[Behavioral Patterns](./Behavioral/)**: Dealing with communication between objects.

## 🛠️ How to use
Each pattern is documented inside its respective category folder as a clean, easy-to-read `.txt` file.
